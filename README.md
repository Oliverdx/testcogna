## Getting Started

First, go to the terminal and install the dependencies from the enviroment:

```bash
npm install
```

run the development server:

```bash
npm run dev
```

After install run and the server be online 
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Run the tests

This application has some unit tests in the components, to run the tests please go to the terminal and type:

```bash
npm run test
```

After the code run all the tests should be passed
